# Senior Planner Polina v2.5

Профессиональный планировщик для специалистов, объединяющий рабочие задачи, учебные трекеры и аналитику состояния тела.

## ✨ Особенности
- **Интерфейс "Glassmorphism"**: Современный дизайн с эффектом матового стекла и динамическими градиентами.
- **Умный Бэклог**: Автоматическое разделение задач на «Ближайшие» (7 дней) и «Когда-нибудь».
- **Голосовой ввод**: Добавление задач голосом прямо из браузера.
- **Аналитика и Heatmap**: Визуальная карта активности (как на GitHub) и графики продуктивности.
- **Учебные трекеры**: Специальные сетки для отслеживания прогресса (например, по работам Фрейда).
- **Drag-and-Drop**: Свободное перемещение задач между колонками.

## 🚀 Быстрый старт
1. Установите **Python 3.12+**.
2. Скачайте проект и запустите `run.bat`.
3. Скрипт сам создаст виртуальное окружение, установит зависимости и запустит сервер.
4. Откройте в браузере: `http://localhost:8000`.

## 🛠 Технологии
- **Backend**: FastAPI (Python), SQLAlchemy (Async), SQLite.
- **Frontend**: Vanilla JS, Chart.js, SortableJS, Glassmorphism CSS.

## 🔒 Конфиденциальность
База данных `planner.db` хранится локально и не попадает в репозиторий. Для инициализации пустой базы используется `init_db.py`.
