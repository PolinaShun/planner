# Протокол Тестирования (Senior Planner Polina)

## Сценарии использования (Актуально на 2026-05-05)

### 1. Основной функционал задач
- [ ] Добавление задачи через поле "Фокус дня" клавишей **Enter**.
- [ ] Перетаскивание задачи между списками (Сегодня -> Бэклог).
- [ ] Отметка выполнения (задача должна затухать и уходить в архив через 0.5с).
- [ ] Редактирование через иконку карандаша.

### 2. Календарь (Мини и Полноэкранный)
- [ ] Мини-календарь: перетаскивается за заголовок, закрывается кликом вне, не вылетает за края экрана.
- [ ] Полноэкранный календарь: задачи отображаются в ячейках дат, переключение месяцев работает.

### 3. Раздел "Прогресс и Аналитика"
- [ ] Метрики тела (Вес, Талия и др.): текст ярко-белый/шалфейный, шрифт системный.
- [ ] Трекер тренировок: сетка компактная (как у Фрейда), отметки сохраняются.
- [ ] Аналитика 2026: графики веса (с 01.01.2026) и прогресса (по месяцам) рендерятся.
- [ ] Архив месяца: сетка за прошлый месяц (Апрель) отображает реальные данные (18/30).

### 4. Визуал и Стабильность
- [ ] Цветовая схема: Smoky Green (дымно-зеленый).
- [ ] Glassmorphism: есть размытие фона и прозрачность карточек.
- [ ] Ошибки: при загрузке страницы (`loadAll`) нет алертов "Ошибка при загрузке".
