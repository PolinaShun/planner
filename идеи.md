# идеи

## glass-flow
- прозрачный интерфейс (glassmorphism) — чтобы выглядело дорого и системно
- трекер состояния: фокус, поток, база, факап
- виджеты для десктопа чтобы мозолили глаза и напоминали про дофамин

## контент (полина v2.0)
- пост про то как дофаминовые петли заставляют нас плодить задачи которые мы не делаем
- системность vs хаос: почему "просто начни" не работает без структуры
- разбор эчв (эмоционально-чувственного восприятия) через призму продуктивности

## проект планировщик
- добавить вкладку с быстрыми идеями (как этот файл но в интерфейсе)
- интеграция с тг-ботом для сброса мыслей на ходу
- архив старых идей чтобы смотреть какой ты был кринж год назад

## разное
- селяви и прочие экзистенциальные штуки для лонгридов
- база по психоанализу (фрейд/лакан) в контексте интерфейсов
